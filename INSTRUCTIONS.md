# Uncovering Strategy Collapse in RL-Trained LLMs

## Overview
This project investigates whether reinforcement learning (RL) reduces reasoning diversity in large language models (LLMs). 

We analyze reasoning trajectories generated by:
- Base models
- Supervised fine-tuned (SFT) models
- RL-trained models (Direct Preference Optimization or PPO)

We convert Chain-of-Thought (CoT) reasoning into **Reasoning Graphs** and evaluate structural differences (Strategy Entropy, Diversity) across models.

## Action Plan (Phase 1)

1.  **Trace Generation**: Sample 10-20 reasoning paths for GSM8K questions using Base, SFT, and RL versions of a model.
2.  **Step Clustering**: Use semantic embeddings to group similar reasoning steps into "states".
3.  **Graph Analysis**: Construct reasoning graphs and calculate:
    - **Strategy Diversity**: Uniqueness of found paths.
    - **Strategy Entropy**: Predictability vs. Variety.
4.  **Midterm Prep**: Generate comparative graphs for specific hard problems to show path narrowing in RL models.
